WHAT MAKES IT DIFFERENT

        

          

            
Semantic, not keyword

            
CLIP embeds understand meaning. "melancholy evening" finds footage that feels that way — not just shots with people outside at night.

          

          

            
Color science, not presets

            
LUTs derived from footage analysis. warm-analog · cool-stark · raw. FFmpeg pipelines, not Instagram filters.

          

          

            
4-signal QA before edit

            
Sharpness, palette coherence, motion energy, brightness — every clip scored before it enters the timeline.

04 — BRAINROT · SIBLING PIPELINE

        
Brainrot.


        
        

          reddit story
           → 
          TTS narration
           → 
          gameplay footage
           → 
          karaoke captions
        


        

          Takes a Reddit micro-story, generates local TTS narration, lays it over split-screen gameplay footage, and burns word-synced karaoke captions. The format that broke TikTok, automated.
        


        

          

            

            

            

            brainrot — zsh
          

          

            
$ python brainrot.py generate --mood funny --telegram

            
  ↳ narration · split-screen · karaoke · $0.01

          

        


      


      

        
WHAT'S DIFFERENT FROM REELGEN

        
Brainrot is a separate pipeline optimized for the Reddit-story format. Local TTS via Kokoro-82M (Apache 2.0) — no ElevenLabs bill. Gameplay footage is provided from your own library.

        
Word-synced karaoke burns are the whole point. Every word highlighted on exact cue, in a font stack that doesn't look like a PowerPoint. Works offline, no API required if you skip the story-polish step.

        

          
MODES

          

            
--mood funny · touchy · self_realization

            
--telegram   (direct-post to Telegram channel)

            
--preview    (see the story, skip rendering)
